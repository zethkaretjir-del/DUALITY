## 🌓 **Two Sides of the Same Coin**

### ⚫ **Duality-Attack** (Penetration Testing Toolkit)
- 🔍 OSINT (Phone, IP, Username, Subdomain, Port, Dork, Email, WHOIS, DNS)
- ⚔️ Attack (Payload Generator, C2, Stealth, Anonymity, DDoS)
- 🤖 Botnet (C2 Server, Web Panel, Discord Bot, Telegram Bot)
- 🛠️ Utility (Crypt, Hash, Encode, Password Gen, Malware, Update)
- 🚀 Advanced (Persistence, File Transfer, Screenshot, Keylogger, Stealer, Miner)

### ⚪ **Duality-Defense** (Security Defense Toolkit)
- 🛡️ Firewall Management
- 👁️ Intrusion Detection System (IDS)
- 🦠 Malware Scanner
- 🪤 Honeypot
- 💾 Backup & Restore
- 📡 Network Monitoring
- 🔐 Encryption Suite
- 📝 Log Analyzer
- 🚨 Incident Response
